Sylius

Framework E-commerce créé par Paweł Jędrzejewski

Ensemble de bundles Symfony et composants PHP e-commerce

Sylius Starndard Edition

Quelques chiffres : ~200 contributeurs / ~1700 stars

Composants PHP

  • Ensemble de composants intégrant le métier
  • Utilisable dans n'importe quel projet PHP (pas couplé à un framework)
  • Peu couplé à d'autres livrairies tierces

Bundles Symfony

  • Ensemble de bundles utilisables indépendement les uns des autres
  • Chaque fonctionnalité e-commerce est disponible dans des bundles différents distribués via composer
  • Peuvent être intégré à une application Symfony existante

Application E-commerce

  • Peut être utilisé tel quel ou peut servir de base à un projet.
  • Composer de deux bundles CoreBundle et le WebBundle
