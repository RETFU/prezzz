Qu'est que la communauté apporte t'elle ?

La communauté

  • La communauté s'est bien diversifié, oui y'a du skill maintent !
  • Beaucoup de projets open source demandent une bonne qualité de code
  • Et ils sont testés !!

PHP Framework Interop Group

  • Regroupe des acteurs majeurs du monde PHP (Sf, Zf, eZ, Doctrine, etc.)
  • Parler des problématiques communes à chaque projet
  • Création des PSRs

Les normes PSR PHP (Specification Request)

  • PSR0 et PSR4 : Chargement des classes
  • PSR1 et PSR2 : Style du code
  • PSR3 : Gestion des log
  • PSR7 : Message HTTP

Industrialisation de PHP

  • Gestionnaire de dépendences : Composer
  • Framework/Micro Framework : Symfony, Laravel, Zend, Yii, Slim, Silex, etc...
  • Framework de tests : PhpUnit, Atom, Behat ou bien PhpSpec.
